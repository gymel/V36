Wenn Sie schon eine Installation haben: 
Wählen Sie das Verzeichnis, wo diese liegt, also kein neues
Verzeichnis anlegen für jede neue Version!
Dann werden nur die neueren Teile überschrieben.
Ihre Daten bleiben stets unberührt! Allenfalls:
WENN Sie Änderungen gemacht haben an:
     $a.cfg, d-1.apr, cat.api, d-wrtf.apr, 
dann kopieren Sie diese auf Ihr Datenverzeichnis, 
falls sie nur auf dem Programmverzeichnis liegen
(also z.B. von C:\ALLGRO nach C:\ALLEGRO\KATALOG,
aber vielleicht liegen sie da schon.)

Die neuen Programme erfordern keine Änderungen an 
existierenden Datenbanken. Die Windows-Programme greifen
unmittelbar auf alte Datenbanken zu, die DOS-Programme
können deshalb immer noch gleichzeitig benutzt
werden - kein entweder/oder! Auch auf Vista kann man die
DOS-Programme uneingeschränkt verwenden.

Falls beim Installieren das Programm a99.exe oder ein anderes
gerade in Benutzung ist, wird am Ende dazu aufgefordert, den
Computer neu zu starten. Erst dann können die neuen Programme
kopiert werden, was dann automatisch geschieht.

Handbuch - nein danke?
Vollständige Dokumentation ist integriert: Dafür gibt es
das "Füllhorn" (oben rechts auf dem Menü).
Wenn Sie die Schriften nicht sofort installieren können:
Diese werden als A-*.TTF auf Ihr Programmverzeichnis kopiert.
Die TTF-Dateien dann später ganz normal installieren.

Für die Win-Programme weisen wir auf folgende Punkte hin: 

- "a99/alcarta" sind monolithische Programme, brauchen 
  also keinen Server. Daher sind sie viel schneller als ein 
  Client/Server-Programm überhaupt sein kann, praktisch 
  so schnell wie DOS-APAC und PRESTO, obwohl sie viel 
  mehr machen.

- Das Anzeigefeld ist in RTF programmierbar. Darin kann 
  markiert und Text über die Zwischenablage in andere 
  Programme übernommen werden. (Strg+a, Strg+c, Strg+v 
  gehen auch). In WinWord kommen alle Attribute herüber, 
  auch die Farben und die Sonderzeichen des allegro-
  Zeichensatzes.

- Die "Links" heißen bei allegro "Flips". Wenn man einen 
  Flip anklickt, kann dahinter 
  (a) ein Find-Befehl stecken (dann erscheint die Ergebnis-
      menge bzw. ein anderer Datensatz), oder
  (b) ein Registersprung (dann erscheint der Register-
      abschnitt) aber auch 
  (c) ein Programmaufruf. Das kann dann auch Netscape
      sein, d.h. ein unmittelbarer Sprung aus a99/alcarta 
      ins Internet ist möglich. Außerdem 
  (d) eine andere Anzeigeform desselben Datensatzes.
      Links können wie bei HTML sehr
      unterschiedlich aussehen - alles Sache der 
      Parametrierung.
  (e) Die mächtigste Möglichkeit ist aber das Auslösen eines
      "Makros" per Flip. Die Makrosprache FLEX ermöglicht
      das Programmieren von Aktionen aller Art, siehe 
      Dokumentation.

- Bei allen allegro-Programmen sind die Menüexte frei
  zugänglich (UIF-Dateien), können also angepaßt werden. 
  Mitgeliefert werden sie auf Deutsch und Englisch 
  (Dateien UIFEGER und UIFEENG)

Das Erstellen neuer Win-Anzeigeparameter ist die einzige
etwas schwierige Arbeit (wenn man nicht das Standard-
system einsetzt), die wir jedoch durch Beispiele und 
Hilfedateien unterstützen.
Siehe Menüpunkt "Installation" im "Füllhorn".

In der Registry werden nur Installationsort und Version
hinterlegt und die Namen der Schriften.


allegro-C Database program package
----------------------------------

"a99" is a Windows based database browsing and editing
program.
It forms part of the "allegro" software package, being used 
in a large number of German libraries for many functions and 
many different databases.
"allegro" is fully configurable, its internal format can be 
MARC but also any other format.

     http://www.alcarta.com

is now the homepage of the latest product line, "alcarta" 
being the new OPAC program and database browser for Windows 
"a99" is the same, but plus the editing functions.
Click on the Union Jack to get the English page, or go 
straight to
     http://www.biblio.tu-bs.de/allegro/alcarta/elcarta.htm


You need: Win'98 or newer, Screen resolution 800x600 
or better.

On the download page, we are making available:
  http://www.allegro-c.de/newvers.htm

1.  The program package 
    Start INST-ALL.EXE on your Windows system. 
    It unwraps to C:\ALLEGRO
    (unless you change that to a different drive in the 
    InstallShield procedure) Three fonts will be installed, 
    containing special characters (A-TIMES.TTF and 
    A-LETTER.TTF for Windows, A-DOS.FNT for DOS)
    Also contained is a small demo database DEMO, 
    installed on C:\ALLEGRO\DEMO2

2.  Several other databases, downloadable from
    http://www.allegro-c.de/alcarta/download.htm


Use F1 for help in all situations, 
plus the big [?] button for database help.
